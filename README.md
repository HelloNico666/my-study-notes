# 📚 学习笔记 - Task01 AI 时代，会说话就会编程

## 📅 日期：2026-02-10

### ✅ 学习内容
- [x] 完整复现 AI 原生贪吃蛇游戏
- [x] 自创 1 个 AI 原生小游戏或 Demo

### 📖 学习心得
以上两个学习内容还相对简单，最大的一个感受是，有时候我们需要知道功能为什么会出现这个错误，直接告诉AI往哪个方向修改会比较容易。举两个我遇到的例子：
1. 在贪吃蛇游戏中，一开始游戏区域的网格在最右边和最下边一行有部分遮挡，导致出现在这两行的食物会显示不全。我跟AI说了两次这个问题，它调整的结果要么是变成最左边和最上边一行出现遮挡，要么就是原来的两行出现遮挡。于是我直接告诉它应该增大游戏区域，这下它就能调整好了。
2. 在自创的小游戏中有个数独的功能，有一次明明已经填写对了，游戏提示答案错误。我不知道是什么原因，就让AI把错误答案标记出来，这下明白了。其实数独不是唯一解的，但AI只以其中一个解来生成正确答案，自然就匹配不上了。
